# コスト見積もり (Cost Estimate)

「GitHub SEO Site」を運用する場合の月額ランニングコストの概算です。低コストを前提としたMVP構成に基づいています。

## 1. ホスティング費用 (Hosting)

**採用サービス: Cloudflare Pages (Free Plan)**

- 月額料金: **$0 / 月**
- **無料枠の制限**:
  - 1アカウントあたりのビルド実行: 500回 / 月（毎日自動生成で約30回/月、手動ビルド含め余裕あり）
  - カスタムドメイン: 無制限
  - 帯域幅: 無料 (Egress Free)
- ※ ページビューが増えすぎてアクセス数が膨大になった場合でも、静的ホスティングからのCDN配信のみであれば基本無料内で運用可能です。

## 2. LLM API 費用 (OpenAI: gpt-4o-mini 等)

高品質かつ低コストなモデル（例: gpt-4o-mini）を想定した試算です。

- **単価 (推定)**: 1000入力トークンあたり約 `$0.00015`、1000出力トークンあたり約 `$0.0006`。

### ケーススタディ: 月300記事生成・20%再生成・README平均4000トークン
このケースでの1ヶ月あたりのLLM費用を試算します。

- **1記事あたりのトークン数とコスト**:
  - 入力（README本文・検索結果など）: 平均 4,000 トークン -> `$0.00060`
  - 出力（解説記事やメタデータ等）: 平均 2,000 トークン -> `$0.00120`
  - 1記事あたりの合計コスト: 約 **`$0.0018`** （約0.27円）
- **月間生成回数**:
  - 新規生成: 300記事
  - 再生成 (20%): 60記事
  - 合計APIコール回数: 360回 / 月
- **月額概算**:
  - 360回 × `$0.0018` = **`$0.648 / 月`**

※ さらに推論能力の高いモデル（gpt-4oなど）を用いる場合、単価が約30倍〜50倍になるため、同条件で月額 `$20〜$30` 程度となります。まずはmini系モデルでの運用を強く推奨します。

## 3. GitHub API 制限と費用

- GitHub REST API を用いたリポジトリ情報の取得。
- **Personal Access Token (PAT) 認証時**: 1時間あたり最大 5,000 リクエスト。
- **月額費用**: **$0** (無料枠内で十分カバー可能)
- 自動収集スクリプトを連続して実行しても、1回あたり数リクエストで済むためRate Limitには到達しません。

## 4. その他必要な費用

- **ドメイン取得・更新費用**: `〜.com`, `〜.dev` などの独自ドメインを利用する場合。
- **年額**: 約 $10 〜 $20 / 年 （月割りで **約 $1 〜 $1.5 / 月**）

---

## 月額コスト合計（最低ライン概算）

- **合計**: **約 $1 〜 $2 / 月**
  - （内訳）ホスティング: $0, LLM API: $0.6, ドメイン代月割: $1〜1.5

個人開発やスモールスタートとしては、この上ない低ランニングコストで運用可能です。広告収益（AdSense等）が月間数ドル入るだけで黒字化できます。
